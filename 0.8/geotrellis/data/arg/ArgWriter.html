geotrellis.data.arg

ArgWriter

case class ArgWriter(typ: RasterType) extends Writer with Product with Serializable

ArgWriter will write a raster to disk in ARG format.

Each instance of ArgWriter is provided a data type (e.g. int or float) to use for output files.
