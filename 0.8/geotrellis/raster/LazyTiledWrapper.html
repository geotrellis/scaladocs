geotrellis.raster

LazyTiledWrapper

case class LazyTiledWrapper(data: ArrayRasterData, tileLayout: TileLayout, rasterExtent: RasterExtent) extends TiledRasterData with Product with Serializable

This RasterData warps an existing ArrayRasterData so as to allow operations on it to be executed in parallel. The underlying RasterData is limited to a single array's worth of data (~2.1G).
