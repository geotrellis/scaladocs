ResampledArrayRasterData

case class ResampledArrayRasterData(underlying: ArrayRasterData, src: RasterExtent, dst: RasterExtent) extends ArrayRasterData with Product with Serializable

This trait represents a raster data which represents a lazily-applied cropping of an underlying raster data object.
