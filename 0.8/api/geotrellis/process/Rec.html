Rec

trait Rec[T] extends AnyRef

Records are the raw scala/json objects, rather than the objects we actually want to pass to the constructors.

Record[T] is expected to implement a create method which builds an instance of T. Records are also required to have a name (which will be used when building maps out of lists.
