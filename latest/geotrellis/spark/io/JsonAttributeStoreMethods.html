JsonAttributeStoreMethods

Related Doc: package io

class JsonAttributeStoreMethods extends AnyRef

Internally we are only using AttributeStore[JsonFormat]. Knowing the format allows us to abstract over writing and reading layer attributes. Layer attributes are almost always required to be read and written together to perform useful work on a layer.

We'd like to be flexible on reading attributes, attempting several fail-overs to account for changes and we'd like to consolidate the writing logic such that attribute stores are as consistent as possible.

WARNING: The fail-over feature is a stop-gap until 0.10 release at which point it will be removed. At that point this class can be used to read all layer attributes for a catalog and write them back, performing an effective migration.
