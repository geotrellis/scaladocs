FileSparkLayerProvider 

class FileSparkLayerProvider extends FileCollectionLayerProvider with LayerReaderProvider with LayerWriterProvider

Provides FileLayerReader instance for URI with file scheme. The uri represents local path to catalog root. ex: file:/tmp/catalog
